# SVU Events Guide

## دليل فعاليات الجامعة الافتراضية السورية

## 1. مقدمة

يهدف مشروع **SVU Events Guide** إلى إنشاء منصة ويب تساعد طلاب الجامعة الافتراضية السورية على استعراض الفعاليات الأكاديمية بطريقة منظمة وسهلة الاستخدام.

تتيح المنصة عرض الفعاليات المختلفة مثل الورشات التعليمية، الندوات، المؤتمرات، والأنشطة الأكاديمية، مع توفير معلومات تفصيلية عن كل فعالية.

---

## 2. فكرة المشروع

تقوم فكرة المشروع على بناء موقع إلكتروني لعرض دليل شامل للفعاليات الجامعية، بحيث يستطيع المستخدم:

* مشاهدة الفعاليات المتاحة.
* تصفح الفعاليات حسب التصنيف.
* الدخول إلى صفحة تفاصيل كل فعالية.
* معرفة معلومات مثل التاريخ، المكان، الوصف، والصور.
* حجز الفعالية.
* إضافة الفعالية إلى التقويم.
* مشاركة رابط الفعالية.

---

## 3. التقنيات المستخدمة

تم تطوير المشروع باستخدام تقنيات تطوير الويب التالية:

### HTML5

استخدام HTML5 لبناء هيكل صفحات الموقع وإنشاء العناصر الأساسية مثل الصفحات، البطاقات، النماذج، والقوائم.

### CSS

استخدام css لتنسيق الموقع وإضافة التصميمات الخاصة مثل:

* تنسيق الألوان والخطوط.
* تصميم البطاقات.
* تحسين مظهر الصفحات.
* دعم الوضع الليلي.

### JavaScript

استخدام JavaScript لإضافة التفاعل والوظائف الديناميكية مثل:

* البحث عن الفعاليات.
* تصفية الفعاليات حسب التصنيف.
* تغيير الوضع إلى Dark Mode.
* زر العودة إلى أعلى الصفحة.
* الحجز وتخزين البيانات.

### Bootstrap 5

تم استخدام Bootstrap 5 عبر **CDN** من خلال تحميل ملفات Bootstrap مباشرة من خادم jsDelivr، دون الحاجة إلى تنزيل الملفات محلياً.

تم استخدام Bootstrap من أجل:

* تصميم واجهات متجاوبة مع مختلف أحجام الشاشات.
* استخدام Navbar.
* استخدام Cards لعرض الفعاليات.
* استخدام Carousel لعرض الفعاليات البارزة.
* استخدام Buttons و Modal.

---

## 4. صفحات المشروع

### الصفحة الرئيسية (index.html)

تحتوي على:

* شريط التنقل.
* Slider للفعاليات البارزة.
* تصنيفات الفعاليات.
* بطاقات أحدث الفعاليات.
* معلومات مختصرة عن كل فعالية.

### صفحة تفاصيل الفعالية (event.html)

تحتوي على:

* صورة الفعالية.
* اسم الفعالية.
* الوصف.
* التاريخ والمكان والتصنيف.
* معرض صور.
* موقع الفعالية على الخريطة.
* إمكانية الحجز.
* إضافة الفعالية إلى التقويم.
* مشاركة الفعالية.

### صفحة عن الدليل (about.html)

تحتوي على:

* تعريف بالمشروع.
* رسالة ورؤية المنصة.
* معلومات عن فريق المشروع.
* سياسات الفعاليات.

### صفحة التواصل (contact.html)

تستخدم لتوفير وسائل التواصل مع إدارة المنصة.

---

## 5. أهم الوظائف المضافة

* عرض الفعاليات بطريقة منظمة باستخدام البطاقات.
* البحث عن الفعاليات.
* تصفية الفعاليات حسب التصنيف.
* الوضع الليلي Dark Mode مع حفظ الاختيار باستخدام LocalStorage.
* زر العودة إلى أعلى الصفحة.
* نظام حجز بسيط للفعاليات باستخدام LocalStorage.
* إضافة الفعاليات إلى Google Calendar.
* مشاركة روابط الفعاليات.
* تصميم متجاوب يعمل على مختلف الأجهزة.
